# MT-PyMOL
MODE-TASK plugin for PyMOL

Install dependencies by runnig the following command

sudo pip install -r requirements.txt

Additionally pyMODE-TASK requires Tkinter and Pmw.1.3. Tkinter comes prepackaged with most standard python. Pmw could be installed by following the instruction from:

http://pmw.sourceforge.net/doc/starting.html
